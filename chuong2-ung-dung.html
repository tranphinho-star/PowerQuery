Một số ứng dụng xử lý dữ liệu thực tế

Khám phá cách Power Query giải quyết các vấn đề dữ liệu thực tế hàng ngày: tổng hợp báo cáo, làm sạch dữ liệu, import đa nguồn và tự động hóa quy trình.

2.1 Tổng hợp báo cáo từ nhiều file Excel

Trong thực tế, dữ liệu thường được lưu trữ trong nhiều file Excel riêng biệt (ví dụ: mỗi tháng một file, mỗi chi nhánh một file). Power Query cho phép bạn gộp tất cả file trong một folder chỉ với vài bước.

Tình huống thực tế

📋
Ví dụ: Bộ phận kế toán nhận báo cáo doanh thu từ 12 chi nhánh, mỗi chi nhánh gửi 1 file Excel hàng tháng. Cần tổng hợp tất cả vào 1 bảng duy nhất để phân tích.

Các bước thực hiện

1

Chuẩn bị thư mục

Đặt tất cả file Excel cần gộp vào cùng một thư mục. Đảm bảo các file có cấu trúc cột giống nhau.

2

Kết nối thư mục

Vào tab Data → Get Data → From File → From Folder. Chọn đường dẫn thư mục chứa các file.

3

Combine & Transform

Nhấn Combine & Transform Data. Power Query sẽ tự động nhận diện cấu trúc và gộp toàn bộ dữ liệu từ tất cả file.

4

Xử lý và Load

Thực hiện các thao tác chuyển đổi cần thiết (lọc, đổi tên cột, thay đổi kiểu dữ liệu) rồi chọn Close & Load.

💡
Mẹo: Khi có file mới được thêm vào thư mục, chỉ cần Refresh query để dữ liệu tự động cập nhật — không cần setup lại từ đầu!

2.2 Làm sạch dữ liệu (Data Cleaning)

Dữ liệu thực tế thường có nhiều vấn đề: khoảng trắng thừa, dữ liệu trùng lặp, giá trị null, định dạng không nhất quán. Power Query cung cấp nhiều công cụ để giải quyết các vấn đề này.

Các vấn đề dữ liệu phổ biến và cách xử lý

Vấn đề Giải pháp trong Power Query Thao tác
Khoảng trắng thừa Trim Transform → Format → Trim
Chữ hoa/thường không nhất quán UPPER / lower / Capitalize Transform → Format → chọn kiểu
Dữ liệu trùng lặp Remove Duplicates Home → Remove Rows → Remove Duplicates
Giá trị null / trống Replace Values / Remove Errors Transform → Replace Values
Sai kiểu dữ liệu Change Type Nhấp vào icon kiểu dữ liệu trên header cột
Cột không cần thiết Remove Columns Home → Remove Columns
Dòng header bị lẫn trong data Use First Row as Headers Transform → Use First Row as Headers

2.3 Import dữ liệu từ Web

Power Query cho phép lấy dữ liệu trực tiếp từ trang web (ví dụ: tỷ giá ngoại tệ, bảng xếp hạng, giá chứng khoán).

1

Kết nối Web

Tab Data → Get Data → From Other Sources → From Web. Nhập URL của trang web chứa bảng dữ liệu.

2

Chọn bảng dữ liệu

Power Query sẽ phân tích trang web và liệt kê các bảng tìm được. Chọn bảng phù hợp và nhấn Transform Data.

3

Xử lý và cập nhật

Thực hiện các bước clean data nếu cần, sau đó Close & Load. Dữ liệu có thể được Refresh để lấy thông tin mới nhất từ web.

2.4 Kết hợp dữ liệu từ nhiều nguồn

Một trong những thế mạnh lớn nhất của Power Query là khả năng kết hợp dữ liệu từ nhiều nguồn khác nhau — ví dụ: gộp dữ liệu từ file Excel với dữ liệu từ SQL Server, hoặc kết hợp thông tin từ CSV với dữ liệu web.

Ví dụ tình huống

📋
Tình huống: Bạn có bảng Đơn hàng trong file Excel (chứa Mã SP, SL, Ngày) và bảng Sản phẩm trong SQL Server (chứa Mã SP, Tên SP, Giá). Cần kết hợp hai bảng để tính doanh thu.

Giải pháp: Sử dụng Merge Queries với cột Mã SP làm khóa nối — tương tự phép JOIN trong SQL. Chi tiết cách thực hiện sẽ được trình bày ở Chương VI: Gộp dữ liệu.

2.5 Tự động cập nhật báo cáo định kỳ

Sau khi thiết lập các bước xử lý, Power Query lưu toàn bộ quy trình. Bạn chỉ cần Refresh để cập nhật dữ liệu mới:

  • Refresh thủ công: Tab Data → Refresh All (hoặc Ctrl + Alt + F5)
  • Refresh khi mở file: Right-click vào query → Connection Properties → đánh dấu "Refresh data when opening the file"
  • Refresh tự động theo chu kỳ: Connection Properties → đánh dấu "Refresh every X minutes"
⚠️
Lưu ý: Refresh tự động chỉ hoạt động khi file Excel đang mở. Đối với refresh nền không cần mở file, cần sử dụng Power Automate hoặc Power BI Service.

2.6 Xử lý dữ liệu từ PDF

Từ Excel 365 và Excel 2021, Power Query hỗ trợ trích xuất bảng dữ liệu trực tiếp từ file PDF — rất hữu ích khi nhận báo cáo dạng PDF.

1

Kết nối PDF

Tab Data → Get Data → From File → From PDF. Chọn file PDF cần import.

2

Chọn bảng

Power Query liệt kê các bảng và trang tìm được trong PDF. Chọn bảng phù hợp.

3

Clean data

Dữ liệu từ PDF thường cần nhiều bước clean: Use First Row as Headers, Remove Columns, Change Type...

Bài tập 2.1: Tổng hợp dữ liệu bán hàng

Mục tiêu: Thực hành tổng hợp dữ liệu từ nhiều sheet trong 1 file Excel

  1. Tạo file Excel tên BanHang2024.xlsx với 3 sheet: Thang1, Thang2, Thang3
  2. Mỗi sheet có cấu trúc: MaSP | TenSP | SoLuong | DonGia | ThanhTien (nhập 5-8 dòng mỗi sheet)
  3. Mở Power Query: Data → Get Data → From File → From Excel Workbook → chọn file
  4. Chọn sheet Thang1 → Transform Data
  5. Trong Power Query Editor: Home → Append Queries → chọn Thang2 và Thang3
  6. Kiểm tra dữ liệu đã gộp đủ 3 tháng
  7. Xóa các dòng trùng lặp (nếu có): Home → Remove Rows → Remove Duplicates
  8. Close & Load để xem kết quả

Bài tập 2.2: Làm sạch dữ liệu khách hàng

Mục tiêu: Thực hành kỹ năng Data Cleaning

  1. Tạo bảng dữ liệu khách hàng với các vấn đề sau:
    • Cột "Họ tên" có khoảng trắng thừa: " Nguyễn Văn A "
    • Cột "Email" có chữ hoa/thường lẫn lộn: "NguyenVanA@Gmail.COM"
    • Cột "SĐT" có giá trị trống ở một số dòng
    • Có 2-3 dòng dữ liệu trùng lặp hoàn toàn
    • Cột "Ngày sinh" ở dạng text, không phải Date
  2. Mở Power Query và thực hiện lần lượt:
    • Trim cột "Họ tên"
    • Lowercase cột "Email"
    • Replace null trong "SĐT" bằng "Chưa có"
    • Remove Duplicates
    • Change Type cột "Ngày sinh" thành Date
  3. Close & Load và kiểm tra kết quả

Bài tập 2.3: Import dữ liệu từ Web

Mục tiêu: Thực hành lấy dữ liệu từ trang web

  1. Mở Excel → tab Data → Get Data → From Other Sources → From Web
  2. Nhập URL: https://vi.wikipedia.org/wiki/Danh_sách_quốc_gia_theo_dân_số
  3. Chọn bảng phù hợp trong Navigator
  4. Trong Power Query Editor:
    • Xóa các cột không cần thiết (giữ lại: STT, Quốc gia, Dân số)
    • Đổi kiểu dữ liệu cột "Dân số" thành Number
    • Sắp xếp theo dân số giảm dần
    • Giữ lại 20 dòng đầu (Home → Keep Rows → Keep Top Rows)
  5. Close & Load
  6. Thử Refresh All (Ctrl + Alt + F5) để cập nhật dữ liệu mới nhất
