Kết nối và cập nhật dữ liệu trong Power Query

Học cách kết nối Power Query với đa dạng nguồn dữ liệu: File, Database, Web, và thiết lập cơ chế cập nhật (Refresh) tự động.

4.1 Tổng quan các nguồn dữ liệu

Power Query hỗ trợ kết nối với hàng chục nguồn dữ liệu khác nhau, được phân thành các nhóm chính:

Hình 4.1: Các nguồn dữ liệu được Power Query hỗ trợ
📁

From File

Excel, CSV, Text, XML, JSON, PDF, Folder

🗄️

From Database

SQL Server, Access, Oracle, MySQL, PostgreSQL

🌐

From Web & Online

Web URL, SharePoint, OData, Dynamics 365

☁️

From Azure & Cloud

Azure SQL, Azure Blob, Azure HDInsight

4.2 Kết nối từ File

4.2.1 Từ File Excel (.xlsx, .xls)

1

Truy cập

Tab Data → Get Data → From File → From Excel Workbook

2

Chọn file và sheet

Duyệt và chọn file Excel → Navigator hiển thị danh sách Sheet và Table. Nhấp để xem trước (Preview). Chọn Transform Data hoặc Load.

💡
Mẹo: Nên chọn Transform Data thay vì Load trực tiếp, để kiểm tra và xử lý dữ liệu trước khi đưa vào Excel.

4.2.2 Từ File CSV / Text

  1. Tab Data → Get Data → From File → From Text/CSV
  2. Chọn file CSV → Power Query tự động nhận diện:
    • Delimiter: Dấu phân cách (phẩy, tab, semicolon...)
    • Encoding: Mã hóa ký tự (UTF-8, Unicode, ANSI...)
    • Data Type Detection: Tự động nhận diện kiểu dữ liệu
  3. Xem trước dữ liệu → chọn Transform Data hoặc Load
⚠️
Lưu ý về Encoding: Với file CSV tiếng Việt, đôi khi bị lỗi font. Hãy thử chọn UTF-8 (65001) trong phần File Origin/Encoding để hiển thị đúng dấu tiếng Việt.

4.2.3 Từ Folder (Nhiều file cùng loại)

Đây là tính năng mạnh mẽ để gộp nhiều file trong một thư mục:

  1. Tab Data → Get Data → From File → From Folder
  2. Chọn đường dẫn thư mục chứa các file
  3. Power Query liệt kê tất cả file → chọn Combine & Transform Data
  4. Chọn file mẫu (sample file) để xác định cấu trúc
  5. Tất cả file sẽ được gộp tự động với cột Source.Name cho biết dữ liệu từ file nào

4.2.4 Từ File JSON

  1. Tab Data → Get Data → From File → From JSON
  2. Chọn file .json → Power Query phân tích cấu trúc
  3. Sử dụng các nút Expand (↔️) để mở rộng các cấp (nested objects/arrays)
  4. Nhấp vào List hoặc Record để drill-down vào dữ liệu lồng nhau

4.3 Kết nối từ Database

4.3.1 SQL Server

1

Kết nối

Tab Data → Get Data → From Database → From SQL Server Database

2

Nhập thông tin

Server name: tên_server hoặc server\instance. Database name (tùy chọn): tên database cụ thể. Có thể nhập SQL statement trực tiếp trong Advanced options.

3

Xác thực

Chọn phương thức xác thực: Windows (tài khoản Windows hiện tại) hoặc Database (nhập username/password SQL).

4

Chọn bảng

Navigator hiển thị danh sách tables và views → chọn bảng cần import → Transform Data.

4.3.2 Microsoft Access

  1. Tab Data → Get Data → From Database → From Microsoft Access Database
  2. Chọn file .accdb hoặc .mdb
  3. Navigator hiển thị Tables và Queries từ Access
  4. Chọn bảng cần import → Transform Data

4.4 Kết nối từ Web

Power Query có thể lấy dữ liệu bảng (table) từ trang web HTML:

  1. Tab Data → Get Data → From Other Sources → From Web
  2. Nhập URL đầy đủ của trang web
  3. Chọn chế độ xác thực: Anonymous (không cần đăng nhập) hoặc Basic/OAuth (cần thông tin đăng nhập)
  4. Power Query phân tích trang và liệt kê các bảng tìm được (Table 0, Table 1...)
  5. Xem trước và chọn bảng phù hợp → Transform Data

4.5 Cập nhật dữ liệu (Refresh)

Sau khi thiết lập query, bạn có thể cập nhật dữ liệu mới bất cứ lúc nào:

Refresh thủ công

Phương pháp Cách thực hiện Phạm vi
Refresh All Tab Data → Refresh All hoặc Ctrl + Alt + F5 Tất cả queries trong workbook
Refresh từng query Right-click vào bảng kết quả → Refresh Chỉ query được chọn
Refresh từ panel Queries & Connections panel → hover query → nhấn 🔄 Chỉ query được chọn

Refresh tự động

  1. Right-click vào bảng kết quả → Table → External Data Properties (hoặc Connection Properties)
  2. Cấu hình:
    • ☑️ Refresh every X minutes: Tự động refresh theo chu kỳ (ví dụ: 30 phút)
    • ☑️ Refresh data when opening the file: Tự động refresh khi mở file Excel
    • ☑️ Enable background refresh: Refresh nền, không chặn thao tác người dùng

4.6 Quản lý Data Source Settings

Khi đường dẫn file thay đổi hoặc cần cập nhật thông tin kết nối:

  1. Tab Data → Get Data → Data Source Settings
  2. Danh sách tất cả nguồn dữ liệu đã kết nối sẽ hiển thị
  3. Chọn nguồn cần chỉnh sửa → Change Source (đổi đường dẫn) hoặc Edit Permissions (đổi quyền truy cập)
  4. Nhấn Clear Permissions nếu muốn xóa thông tin xác thực đã lưu
💡
Mẹo: Khi chia sẻ file Excel có Power Query cho đồng nghiệp, đảm bảo rằng họ cũng có quyền truy cập vào các nguồn dữ liệu (file đường dẫn, database, SharePoint...).

4.7 Kết nối từ PDF

Power Query có thể trích xuất bảng dữ liệu từ file PDF:

  1. Tab Data → Get Data → From File → From PDF
  2. Chọn file PDF
  3. Navigator hiển thị danh sách Tables và Pages tìm được
  4. Chọn bảng phù hợp → Transform Data
⚠️
Lưu ý: Chất lượng import từ PDF phụ thuộc vào cấu trúc file PDF. File PDF tạo từ scan ảnh sẽ không thể trích xuất — chỉ PDF có text thực mới hoạt động. Cần kiểm tra và làm sạch dữ liệu kỹ sau khi import.

4.8 Kết nối từ XML

  1. Tab Data → Get Data → From File → From XML
  2. Chọn file .xml
  3. Power Query phân tích cấu trúc phân cấp (hierarchical)
  4. Sử dụng các nút Expand (↔️) để mở rộng nested elements
  5. Nhấp vào Table để drill-down vào child elements

4.9 Kết nối từ SharePoint

SharePoint List

  1. Tab Data → Get Data → From Online Services → From SharePoint List
  2. Nhập URL site SharePoint (ví dụ: https://company.sharepoint.com/sites/mysite)
  3. Xác thực bằng tài khoản Microsoft 365
  4. Navigator hiển thị tất cả lists → chọn list cần import

SharePoint Folder

  1. Tab Data → Get Data → From File → From SharePoint Folder
  2. Nhập URL site SharePoint
  3. Power Query liệt kê tất cả file trong document library
  4. Lọc và Combine files tương tự From Folder

OData Feed

  1. Tab Data → Get Data → From Other Sources → From OData Feed
  2. Nhập URL endpoint OData
  3. Xác thực nếu cần
  4. Navigator hiển thị các entities → chọn và Transform Data

4.10 Native SQL Query

Khi kết nối từ database, bạn có thể viết SQL trực tiếp thay vì chọn toàn bộ bảng:

1

Mở Advanced Options

Trong dialog kết nối SQL Server, mở rộng Advanced options

2

Nhập SQL Statement

Viết câu SQL trong ô SQL statement:

SQL
SELECT MaSP, TenSP, DonGia, SoLuong
FROM SanPham
WHERE DanhMuc = N'Máy tính'
ORDER BY DonGia DESC
💡
Ưu điểm: Native SQL Query tận dụng Query Folding — server thực thi SQL và chỉ trả về kết quả, giúp tăng hiệu suất đáng kể với database lớn.

4.11 Query Folding Indicator

Trong Power Query Editor, bạn có thể kiểm tra xem bước nào được fold (đẩy về server xử lý):

Icon Applied StepsÝ nghĩa
✅ Folded (có icon xanh) Bước được thực thi trên server — hiệu suất cao
⚠️ Not Folded (icon vàng) Bước xử lý cục bộ — có thể chậm với dữ liệu lớn
❓ Unknown Không xác định — phụ thuộc nguồn dữ liệu

Bật indicator: View → Query Folding Indicators (có trong Excel 365/Power BI mới nhất).

4.12 Fixed Width Files (File cố định cột)

Một số file text không có delimiter mà sử dụng độ rộng cố định cho mỗi cột:

  1. Data → Get Data → From File → From Text/CSV
  2. Trong dialog Preview, chọn Delimiter = Fixed Width
  3. Power Query tự detect vị trí cột, hoặc bạn chỉnh bằng cách kéo đường phân cách
  4. Transform Data để tinh chỉnh
M Language - Fixed Width
// Import file fixed width thủ công
= Table.SplitColumn(
    Source, "Column1",
    Splitter.SplitTextByPositions({0, 10, 30, 45}),
    {"MaNV", "HoTen", "PhongBan", "Luong"}
)

4.13 Encoding khi import Text/CSV

Khi file CSV chứa ký tự đặc biệt (tiếng Việt, tiếng Nhật...), chọn đúng Encoding rất quan trọng:

EncodingCode PageKhi nào dùng
UTF-865001Chuẩn quốc tế, hỗ trợ tiếng Việt ✅
Windows-12521252File từ Windows cũ, không dấu
UTF-16 LE1200File Unicode từ Notepad Windows
ANSI1258File tiếng Việt cũ (VNI, TCVN)
M Language - Chỉ định Encoding
= Csv.Document(
    File.Contents("D:\Data\DuLieu.csv"),
    [Delimiter=",", Encoding=65001, QuoteStyle=QuoteStyle.Csv]
)

4.14 Dynamic Named Range trong Excel

Khi nguồn dữ liệu trong Excel không phải Table, bạn có thể dùng Named Range hoặc truy cập sheet trực tiếp:

M Language - Truy cập Sheet/Named Range
// Lấy dữ liệu từ Sheet cụ thể
= Excel.Workbook(File.Contents("D:\Data.xlsx")){[Item="Sheet1", Kind="Sheet"]}[Data]

// Lấy dữ liệu từ Named Range
= Excel.Workbook(File.Contents("D:\Data.xlsx")){[Item="DuLieuBanHang", Kind="DefinedName"]}[Data]

// Lấy dữ liệu từ Table
= Excel.CurrentWorkbook(){[Name="BangNhanVien"]}[Content]
💡
Best Practice: Luôn đặt dữ liệu Excel vào Table (Ctrl+T) trước khi import. Table tự mở rộng khi thêm dòng, đảm bảo refresh luôn lấy đủ dữ liệu.

4.15 Web Scraping nâng cao

Power Query hỗ trợ nhiều cách lấy dữ liệu từ web:

Web.Page vs Web.BrowserContents

HàmKhác biệtKhi nào dùng
Web.Page Lấy HTML tĩnh, parse thành bảng tự động Trang web đơn giản có bảng HTML
Web.BrowserContents Render JavaScript trước khi lấy HTML Trang web động (React, Angular, SPA)
Web.Contents Lấy raw data (JSON, XML, text) REST API, file download
M Language - Web Scraping
// Lấy bảng HTML từ trang web
let
    Source = Web.Page(Web.Contents("https://example.com/data")),
    // Chọn bảng thứ 2 trên trang
    Data = Source{1}[Data]
in
    Data

// Lấy HTML render JavaScript
let
    Html = Web.BrowserContents("https://example.com/spa-page"),
    Source = Web.Page(Html),
    Data = Source{0}[Data]
in
    Data

4.16 ODBC & OLEDB Connections

Kết nối với hệ thống cũ hoặc database đặc biệt qua ODBC/OLEDB:

  1. Data → Get Data → From Other Sources → From ODBC (hoặc OLEDB)
  2. Chọn DSN (Data Source Name) đã cấu hình, hoặc nhập Connection String
  3. Xác thực → Navigator → chọn bảng
M Language - ODBC Connection
// Kết nối ODBC với Connection String
= Odbc.DataSource(
    "Driver={MySQL ODBC 8.0};Server=localhost;Database=mydb;",
    [HierarchicalNavigation=true]
)

// Kết nối OLEDB (Access, Oracle...)
= OleDb.DataSource(
    "Provider=Microsoft.ACE.OLEDB.12.0;Data Source=D:\Data.accdb;"
)

4.11 Load Options (Tùy chọn tải dữ liệu)

Khi Close & Load, bạn có 4 lựa chọn về cách tải dữ liệu:

OptionMô tảKhi nào dùng
TableTải vào worksheet dạng TableXem/chỉnh sửa dữ liệu trực tiếp
PivotTableTải trực tiếp vào PivotTablePhân tích tổng hợp ngay
PivotChartTải vào PivotChartTrực quan hóa dữ liệu
Connection OnlyChỉ tạo kết nối, KHÔNG tải vào sheetStaging queries, query trung gian

Cách chọn: Home → Close & Load → Close & Load To...

4.12 Staging Queries (Query trung gian)

Trong thiết kế ETL chuyên nghiệp, bạn tạo Staging Queries (Connection Only) làm nguồn cho các query khác:

Ví dụ Staging Pattern
// stg_DonHang (Connection Only — không tải vào sheet)
let Source = Csv.Document(File.Contents(pFolder & "DonHang.csv"),
    [Delimiter=",", Encoding=65001]),
    Headers = Table.PromoteHeaders(Source),
    Typed = Table.TransformColumnTypes(Headers, {
        {"NgayDat", type date}, {"SoLuong", Int64.Type}
    })
in Typed

// rpt_DoanhThu (Load to Table — dùng stg_DonHang)
let Source = stg_DonHang,
    Grouped = Table.Group(Source, {"ThangDat"},
        {{"TongDoanhThu", each List.Sum([SoLuong]), type number}})
in Grouped
💡
Lợi ích Staging: Giảm trùng lặp code, dễ bảo trì, tối ưu hiệu năng (chuẩn hóa 1 lần, dùng nhiều nơi). Đặt tên prefix stg_ để phân biệt.

Bài tập 4.1: Kết nối từ CSV

Mục tiêu: Thực hành import file CSV vào Power Query

  1. Tạo file SanPham.csv bằng Notepad với nội dung: 
    MaSP,TenSP,DonGia,SoLuong,DanhMuc
SP001,Laptop Dell,15000000,50,Máy tính
SP002,Chuột Logitech,350000,200,Phụ kiện
SP003,Bàn phím Corsair,1200000,100,Phụ kiện
SP004,Màn hình Samsung,5500000,75,Màn hình
SP005,Tai nghe Sony,2800000,150,Phụ kiện
  2. Lưu file với encoding UTF-8
  3. Mở Excel → Data → Get Data → From File → From Text/CSV
  4. Chọn file → kiểm tra Delimiter = Comma, Encoding = UTF-8
  5. Transform Data → kiểm tra dữ liệu trong Power Query Editor
  6. Đổi kiểu dữ liệu: DonGia và SoLuong → Whole Number
  7. Close & Load

Bài tập 4.2: Kết nối từ nhiều file trong folder

Mục tiêu: Thực hành Combine Files from Folder

  1. Tạo thư mục DoanhThu trên Desktop
  2. Tạo 3 file Excel trong thư mục, mỗi file có cùng cấu trúc:
    • DoanhThu_T1.xlsx: Cột MaCH, TenCH, DoanhThu (5 dòng)
    • DoanhThu_T2.xlsx: Cùng cấu trúc (5 dòng khác)
    • DoanhThu_T3.xlsx: Cùng cấu trúc (5 dòng khác)
  3. Mở Excel mới → Data → Get Data → From File → From Folder
  4. Chọn thư mục DoanhThu → nhấn Combine & Transform Data
  5. Quan sát cột Source.Name cho biết dữ liệu từ file nào
  6. Close & Load → kiểm tra dữ liệu 15 dòng (5 × 3 file)
  7. Thử thêm file thứ 4: Tạo DoanhThu_T4.xlsx → đặt vào thư mục → Refresh All → dữ liệu tự động cập nhật!

Bài tập 4.3: Thiết lập Auto Refresh

Mục tiêu: Cấu hình tự động cập nhật dữ liệu

  1. Sử dụng query từ bài tập 4.1 hoặc 4.2
  2. Right-click vào bảng kết quả trong Excel
  3. Chọn Table → External Data Properties
  4. Đánh dấu ☑️ Refresh data when opening the file
  5. Đánh dấu ☑️ Refresh every → nhập 5 minutes
  6. Nhấn OK → đóng và mở lại file → quan sát dữ liệu tự động refresh
