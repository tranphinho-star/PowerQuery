Import CSV tiếng Việt & Khám phá giao diện

Ch.I–III

Đề bài: Tạo file DanhSachNhanVien.csv (UTF-8) gồm các cột: MaNV, HoTen, PhongBan, NgaySinh, Luong (10 dòng). Import vào Power Query, khám phá giao diện, đổi tên cột và set đúng kiểu dữ liệu.

MaNVHoTenPhongBanNgaySinhLuong
NV01Nguyễn Văn AnIT15/03/199018000000
NV02Trần Thị BìnhKế toán22/07/198815000000
NV03Lê Hoàng ChâuIT01/11/199522000000

(Thêm 7 dòng nữa với các phòng ban: "Kinh doanh", "Nhân sự", "Marketing")

Gợi ý:
1. Dùng Notepad → Save as → Encoding: UTF-8
2. Data → Get Data → From File → From Text/CSV
3. Chọn Delimiter = Comma, Character Set = 65001 (UTF-8)
4. Trong PQ Editor: khám phá ribbon (Home, Transform, Add Column), Query Settings panel, Formula Bar
Đáp án: 
let
    Source = Csv.Document(
        File.Contents("D:\Data\DanhSachNhanVien.csv"),
        [Delimiter=",", Encoding=65001]
    ),
    Headers = Table.PromoteHeaders(Source),
    SetTypes = Table.TransformColumnTypes(Headers, {
        {"MaNV", type text}, {"HoTen", type text},
        {"PhongBan", type text},
        {"NgaySinh", type date}, {"Luong", Currency.Type}
    })
in SetTypes

✅ Kết quả: 10 dòng, 5 cột, hiển thị đúng tiếng Việt, NgaySinh là Date, Luong là Currency.

Đề bài: Bảng KhachHang có cột "HoTen" (VD: "Nguyễn Văn An") và cột "DiaChi" (VD: "123 Lê Lợi, Quận 1, TP.HCM"). Tách: HoTen → Ho + TenDem + Ten; DiaChi → SoNha + Quan + ThanhPho.

1. Split Column → By Delimiter (Space) cho HoTen
2. Split Column → By Delimiter (Comma) cho DiaChi
3. Rename lại các cột sau khi tách
4. Dùng Text.Trim sau khi split bằng dấu phẩy 
// Tách HoTen bằng khoảng trắng

// Lưu ý: "true" = split from right → "Nguyễn Văn" + "An"

// Tách DiaChi bằng dấu phẩy
= Table.SplitColumn(Source, "DiaChi",
    Splitter.SplitTextByDelimiter(","),
    {"SoNha", "Quan", "ThanhPho"})

// Trim khoảng trắng thừa
= Table.TransformColumns(Prev, {
    {"Quan", Text.Trim}, {"ThanhPho", Text.Trim}

List.Accumulate: Multi-Replace & Calendar Table

Ch.IX

Đề bài: (A) Chuẩn hóa tên: thay thế nhiều ký tự đặc biệt ("–" → "-", " " → " ", "." → "") bằng 1 lệnh. (B) Tạo Calendar Table từ 01/01/2024 đến 31/12/2024 với cột: Ngày, Năm, Tháng, Quý, Thứ, CuốiTuần.

(B) List.Dates + Table.FromList + Date functions 
// (A) Multi-Replace
= Table.AddColumn(Source, "TenClean", each
    List.Accumulate(
        {{"–","-"}, {"  "," "}, {".",""}, {"'",""}},
        [TenSP],
        (state, pair) => Text.Replace(state, pair{0}, pair{1})
    ))

// (B) Calendar Table
let
    Start = #date(2024,1,1), End = #date(2024,12,31),
    Days = Duration.Days(End - Start) + 1,

    AddThang = Table.AddColumn(AddNam, "Thang", each Date.Month([Ngay])),

    AddCT = Table.AddColumn(AddThu, "CuoiTuan",

Làm sạch dữ liệu khách hàng

Ch.X

Đề bài: Bảng KhachHang có dữ liệu bẩn: khoảng trắng thừa, chữ hoa/thường lộn xộn, số điện thoại nhiều format (0901234567, +84901234567, 090-123-4567), dòng trùng, null. Chuẩn hóa tất cả.

1. Text.Trim, Text.Clean cho khoảng trắng
2. Text.Proper cho chữ hoa/thường
3. Text.Replace để chuẩn hóa SĐT
4. Table.Distinct, Table.SelectRows(not null) 
let
    Source = Excel.CurrentWorkbook(){[Name="KhachHang"]}[Content],
    // Xóa null & dòng trống
    NoNull = Table.SelectRows(Source,
        each [HoTen] <> null and [HoTen] <> ""),
    // Trim + Clean
    Cleaned = Table.TransformColumns(NoNull, {
        {"HoTen", each Text.Proper(Text.Trim(Text.Clean(_)))},
        {"Email", each Text.Lower(Text.Trim(_))}
    }),
    // Chuẩn hóa SĐT
    FixPhone = Table.TransformColumns(Cleaned, {
        {"SDT", each Text.Replace(
            Text.Replace(
                Text.Replace(_, "-", ""), "+84", "0"
            ), " ", ""
        )}
    }),
    // Loại trùng theo Email
    NoDups = Table.Distinct(FixPhone, {"Email"})
in NoDups

VBA Auto-Refresh & Best Practices tổng kết

Ch.XIII

Đề bài: (A) Tạo macro VBA tự động refresh khi mở file + timer 30 phút. (B) Áp dụng Naming Convention & Documentation cho toàn bộ project Bài 19: prefix queries, description, comments, rename steps.

(A) Alt+F11 → ThisWorkbook → Workbook_Open
(B) Prefix: raw_, stg_, fn; Description qua Properties 
' (A) VBA - ThisWorkbook module
Private Sub Workbook_Open()
    ' Refresh tất cả khi mở file
    ThisWorkbook.RefreshAll
    ' Khởi động timer 30 phút
    Call StartTimer
End Sub

Sub StartTimer()
    Application.OnTime Now + TimeValue("00:30:00"), "AutoRefresh"
End Sub

Sub AutoRefresh()
    ThisWorkbook.RefreshAll
    Call StartTimer  ' Lập lại timer
End Sub

Sub StopTimer()
    On Error Resume Next
    Application.OnTime Now + TimeValue("00:30:00"), "AutoRefresh", , False
End Sub

(B) Naming Convention checklist:

  • ✅ raw_DonHang, raw_SanPham (prefix "raw_")
  • ✅ stg_DonHang, stg_SanPham (prefix "stg_")
  • ✅ ChiTietDonHang, ThongKeThang (tên rõ ràng)
  • ✅ pFolder, pNam (prefix "p" cho parameters)
  • ✅ fnThueTNCN (prefix "fn" cho functions)
  • ✅ Mỗi query có Description (Properties)
  • ✅ Steps đổi tên: "Changed Type" → "SetColumnTypes"
  • ✅ Comments: // Mục đích bước này

🏆 Final Project: ETL Báo cáo Doanh thu End-to-End

Tổng hợp

Đề bài: Xây dựng hệ thống ETL hoàn chỉnh cho dữ liệu bán hàng:

  • Nguồn 1: Folder chứa 12 file CSV doanh số hàng tháng (cấu trúc giống nhau)
  • Nguồn 2: File Excel danh mục sản phẩm (MaSP, TenSP, DanhMuc, DonGia)
  • Nguồn 3: File Excel danh sách nhân viên bán hàng (MaNV, HoTen, ChiNhanh)

Yêu cầu:

  1. Import từ Folder + giữ tên file (Preserving Context)
  2. Làm sạch: trim, remove errors, chuẩn hóa MaSP
  3. Merge với Sản phẩm + Nhân viên (Left Outer Join)
  4. Thêm cột: ThanhTien, Thang, Quy, NhomGia (Age Banding)
  5. Tạo 3 output: ChiTietDonHang, ThongKeThang, Top10SP
  6. Tổ chức: Groups, Naming Convention, Disable Load cho staging
Gợi ý theo 3-Layer ETL:
01_Raw: raw_DonSo (Folder), raw_SanPham, raw_NhanVien → Connection Only
02_Staging: stg_DonSo (Clean+Context), stg_SanPham, stg_NhanVien → Connection Only
03_Output: ChiTietDonHang, ThongKeThang, Top10SP → Load to Table 
// [01_Raw] raw_DonSo — Connection Only
let Source = Folder.Files(pFolder & "DoanhSo"),
    Filter = Table.SelectRows(Source, each Text.EndsWith([Name],".csv")),
    Add = Table.AddColumn(Filter, "Data",
        each Csv.Document([Content],[Delimiter=",",Encoding=65001])),
    Select = Table.SelectColumns(Add, {"Name","Data"})
in Select

// [02_Staging] stg_DonSo — Connection Only
let Source = raw_DonSo,
    Exp = Table.ExpandTableColumn(Source, "Data",
        Table.ColumnNames(Source{0}[Data])),
    Headers = Table.PromoteHeaders(Exp),
    Clean = Table.TransformColumns(Headers,
        {{"MaSP", Text.Trim}, {"MaNV", Text.Trim}}),
    Typed = Table.TransformColumnTypes(Clean,
        {{"SoLuong",Int64.Type},{"NgayBan",type date}}),
    NoErr = Table.RemoveRowsWithErrors(Typed),
    Month = Table.AddColumn(NoErr, "Thang",
        each Date.Month([NgayBan]))
in Month

// [03_Output] ChiTietDonHang — Load to Table
let S = stg_DonSo,
    M1 = Table.NestedJoin(S,"MaSP",stg_SanPham,"MaSP","SP",JoinKind.LeftOuter),
    E1 = Table.ExpandTableColumn(M1,"SP",{"TenSP","DanhMuc","DonGia"}),
    M2 = Table.NestedJoin(E1,"MaNV",stg_NhanVien,"MaNV","NV",JoinKind.LeftOuter),
    E2 = Table.ExpandTableColumn(M2,"NV",{"HoTen","ChiNhanh"}),
    TT = Table.AddColumn(E2,"ThanhTien",each [SoLuong]*[DonGia]),
    Quy = Table.AddColumn(TT,"Quy",
        each "Q"&Text.From(Date.QuarterOfYear([NgayBan]))),
    Band = Table.AddColumn(Quy,"NhomGia",each
        if [DonGia]<500000 then "Thấp"
        else if [DonGia]<2000000 then "Trung bình"
        else "Cao")
in Band

// [03_Output] Top10SP — Load to Table
let G = Table.Group(ChiTietDonHang,"TenSP",
        {{"TongDoanhThu",each List.Sum([ThanhTien])}}),
    S = Table.Sort(G,{{"TongDoanhThu",Order.Descending}})
in Table.FirstN(S, 10)
